About Ducky

Ducky is a dynamic webpage creator that allows users to generate beautiful static web pages with customizable themes.

It provides a user-friendly interface where you can input your requirements, and it will generate the entire HTML, CSS, and JavaScript code needed for your website.

Whether you're building a personal portfolio, a blog, or any other type of static site, Ducky makes it easy and fast!
